## Sobre la web
Para correr en un servidor de desarrollo:
1. Clonar repositorio
2. Instalar dependencias
```bash
npm install
```
3. Correr proyecto
```bash
npm run dev
```
4. Abrir proyecto en [http://localhost:3000](http://localhost:3000)
